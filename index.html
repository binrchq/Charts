📚 Available Charts

HeadCNI

v1.0.0
k8s 1.20+ Tailscale v1.100.0

Kubernetes Container Network Interface (CNI) plugin integrating Headscale and Tailscale for highly secure, encrypted Pod-to-Pod network fabrics via WireGuard.

  • Zero Trust network encryption with WireGuard
  • Optimized default MTU (1400) for hybrid environments
  • Custom route tables and marks preventing host overlaps
  • Built-in health monitoring and status metrics
helm install headcni binrc/headcni
